# 📊 Dashboard NPS

Sistema web desenvolvido para visualização e análise dos resultados de satisfação dos clientes através da metodologia NPS (Net Promoter Score).

A aplicação transforma os dados coletados na pesquisa de satisfação em gráficos e indicadores visuais, facilitando a tomada de decisão e o acompanhamento da experiência dos clientes.

O dashboard apresenta informações em tempo real com visualização intuitiva e moderna.

✔️ Apoio na tomada de decisão  
✔️ Monitoramento em tempo real  

---

## 📊 Sobre o NPS

O Net Promoter Score (NPS) é um indicador utilizado para medir a satisfação e fidelização dos clientes.

Classificação das respostas:

- **Promotores** → notas 9 e 10  
- **Neutros** → notas 7 e 8  
- **Detratores** → notas de 0 a 6  

Fórmula:

NPS = % Promotores - % Detratores
